- Virtuals Protocol은 Agent Commerce Protocol (ACPRouter + 5개 모듈), TypeScript SDK (acp-node), CLI 마켓플레이스 레퍼런스 (openclaw-acp)로 구성됩니다.
- 유사 마켓플레이스로 Agent Exchange, MarketClaw, SingularityNET, Fetch.ai uAgents가 있습니다.
- 웹 마켓플레이스 레퍼런스로 OpenStock과 OneStopShop이 있습니다.
- 아키텍처 설계에 스마트 컨트랙트 ACP 패턴을 직접 참조해야 합니다.
- 스마트 컨트랙트는 ACPRouter, AccountManager, JobManager, MemoManager, PaymentManager 모듈로 구성됩니다.
- Job 상태 머신은 REQUEST, NEGOTIATION, TRANSACTION, EVALUATION, COMPLETED 단계를 포함합니다.
- 이중 에스크로 시스템은 Job 레벨(budget)과 Memo 레벨(건별 결제)로 운영됩니다.
- 수수료는 platformFeeBP와 evaluatorFee를 포함합니다.
