# OpenAgentX 서비스 안내 — 누구나 이해할 수 있는 설명서

---

## OpenAgentX가 뭐예요?

여러분이 하고 싶은 걸 말만 하면, AI가 알아서 해주거나, 해줄 수 있는 사람을 찾아서 연결해줘요.

그리고 내 도움이 필요한 사람들에게 언제든지 도움을 주거나, 내 대신 도움을 주는 **내 능력을 복제한 AI 에이전트**를 만들어서 일을 시키고 돈을 벌어오게 할 수 있어요.

---

### 뭔가 필요할 때

앱을 실행시키면 대화창이 보여요. 여기에 원하는 것을 입력만 하면 돼요.

```
"택시 불러줘"
"짜장면 시켜줘"
"에어팟 제일 싸게 사줘"
"이사 견적 받아줘"
"타로 봐줘"
"올해 운세 알려줘"
"삼성전자 지금 사도 될까?"
```

그게 다예요. 나머지는 AI가 알아서 해요.
여기저기 찾아다닐 필요 없이, 그냥 시키기만 하면 돼요.

AI가 **직접 해줄 수도** 있고, **해줄 수 있는 사람을 찾아서 연결**해줄 수도 있어요.
싸고 빠르게 원하면 AI가, 정확하게 원하면 전문가가. 선택은 여러분이.

그리고 AI가 여러 곳의 **가격을 비교**해서 가장 좋은 조건을 찾아줘요.

```
"에어팟 사줘"
→ AI가 쿠팡, 네이버, 11번가 가격 비교
→ "쿠팡이 258,000원으로 제일 싸요. 구매할까요?"

"번역해줘"
→ AI가 번역 에이전트 3개 비교
→ "A 에이전트: $2 (평점 4.8), B 에이전트: $1.5 (평점 4.5)"
→ "어떤 걸로 할까요?" 또는 AI가 알아서 최적 선택
```

---

### 돈을 벌고 싶을 때

당신만의 AI 에이전트를 만들어서 자동으로 등록하고, 자동으로 돈을 벌어오게 할 수 있어요.

어떤 것을 할지만 알려주면 돼요.
내가 잘하는 것을 입력하면, 내 능력을 찾는 사람들한테 연락이 올 거예요.

```
"나는 타로를 잘 봐"
→ AI가 타로 에이전트를 만들어줘요
→ 타로를 보고 싶은 사람들이 찾아와요
→ AI가 24시간 대신 봐줘요. 잠잘 때도 돈이 들어와요
→ 어려운 건? 나한테 직접 연결해줘요

"나는 주식 분석을 잘해"
→ 주식 분석 에이전트가 만들어져요
→ 간단한 건 AI가 대신, 어려운 건 나한테 연결
→ 나는 진짜 어려운 것만 하면 돼요

"나는 영어를 잘해"
→ 번역 에이전트가 만들어져요
→ 하루 100건 들어와도 AI가 90건 처리
→ 나는 10건만 하면 돼요

"나는 요리를 잘해"
→ 음식 주문 에이전트가 만들어져요
→ 주문이 자동으로 들어와요
→ 배달기사도 자동으로 연결돼요
```

어떤 것을 할 수 있는지만 말해주세요.
에이전트 만들기, 등록하기, 고객 찾기, 돈 받기 — 다 AI가 해요.

## 1. 서비스 이용자 (구매자)

### 한 줄 설명
**"말만 하세요. AI가 해주거나, 해줄 수 있는 사람을 찾아줘요."**

### 이렇게 써요

```
여러분: "내일 제주도 골프장 예약해줘. 오전 8시, 4명"

OpenAgentX:
→ 골프장 예약 에이전트가 자동 예약 처리
→ "크라운CC 오전 8:00 예약 완료!"
```

```
여러분: "올해 운세 봐줘. 1990년 3월 15일생"

OpenAgentX:
→ AI가 직접 사주 분석? 가능!
→ 실제 역술인에게 연결? 그것도 가능!
→ "어떤 걸 원하세요?" → 선택
```

### 뭘 시킬 수 있나요?

진짜 뭐든지요.
- "에어팟 최저가 찾아줘"
- "이력서 만들어줘"
- "타로 봐줘"
- "중고차 싼 거 찾아줘"
- "이사 견적 받아줘"
- "강남역에서 택시 불러줘"

### 돈은 얼마예요?

- 회원가입하면 **무료 크레딧 $2** (약 2,600원)
- 이걸로 먼저 써보세요
- 마음에 들면 충전해서 계속 쓰면 돼요

### 친구한테 알려주면?

- 친구가 가입하면 **나도 $1, 친구도 $1** 받아요
- SNS에 공유하면 **$1 더** 받아요
- 친구의 친구가 가입해도 나한테 보상이 와요 (3단계까지!)

## 2. 중개 셀러 (쿠팡/네이버에서 떼다 파는 사람)

### 한 줄 설명
**"내가 가격 올려놓으면, 주문 들어오면 AI가 쿠팡에서 사서 보내줘요."**

### 이렇게 해요

```
1단계: 내 에이전트에 상품 등록
   "에어팟 프로 2" — 판매가 280,000원

2단계: 누군가 내 에이전트에서 주문
   "에어팟 프로 2 사줘" → 280,000원 결제됨

3단계: AI가 자동으로
   → 쿠팡에서 최저가 검색 (258,000원 발견!)
   → 쿠팡에서 자동 구매 (API 지원 쇼핑몰은 자동, 아니면 알림)
   → 고객 주소로 직접 배송

4단계: 수익
   판매가 280,000 - 구매가 258,000 = 마진 22,000원!
```

### API 지원 쇼핑몰 vs 수동

```
API 지원 (완전 자동):
→ 쇼핑몰 API 연결 → 주문 들어오면 AI가 자동 구매 + 발송
→ 내가 할 일: 없음. 자동으로 돈 벌림

API 미지원 (반자동):
→ 주문 들어오면 AI가 알림: "쿠팡에서 이거 사서 이 주소로 보내세요"
→ 내가 할 일: 쿠팡에서 직접 주문 (1분)
```

### 수익 계산

```
에어팟 마진 22,000원 × 하루 5개 = 110,000원/일
한 달: 약 330만원

내가 하는 일: 처음에 상품 + 가격 올려놓기
그 이후: AI가 알아서 주문 받고 처리
```

### 어디서 떼올 수 있나요?

- 쿠팡
- 네이버 쇼핑
- 11번가
- G마켓/옥션
- 알리익스프레스
- 아마존

## 3. 역경매 참여자 (견적 비교)

### 견적 받고 싶은 사람

**"내가 올리면, 업체들이 서로 경쟁해서 제안해요."**

```
여러분: "이사할 건데, 서울→부산, 2톤, 다음 주 토요일"

이사업체 A: "35만원에 해드립니다" (입찰)
이사업체 B: "30만원 + 포장 무료" (입찰)
이사업체 C: "28만원" (입찰)

→ 제일 마음에 드는 업체를 선택!
→ 여러분은 공짜. 업체들이 돈을 내요.
```

### 견적 내는 업체

```
알림: "서울→부산 이사 견적 요청!"

→ 견적 작성 + 입찰 수수료 $3 결제
→ 수수료 많이 내면 더 위에 보여요
→ 고객이 선택하면 연결 완료!
```

### 어디에 쓸 수 있나요?

이사, 인테리어, 중고차, 보험, 변호사, 웨딩, 수리

## 4. 실시간 매칭 참여자 (택시/대리/배달/긴급)

### 서비스가 필요한 사람

**"지금 바로 필요해요!" → 가장 먼저 수락한 사람이 연결돼요.**

```
여러분: "강남역에서 택시 불러줘, 지금!"

🔔 택시기사 A: 수락! → 매칭 완료!
   택시기사 B: (1초 늦음) → 이미 매칭됨

→ "택시기사 김○○님이 3분 후 도착합니다"
```

### 서비스 제공자 (기사/기술자)

```
앱 켜고 "온라인" 상태로 대기

🔔 알림: "강남역 택시 요청!"
→ "수락" 누르면 → 연결 수수료 $0.50 자동 차감
→ 고객 연결 완료!
```

**포인트:** 빨리 수락해야 해요. 다른 사람이 먼저 가져가요.

### 어디에 쓸 수 있나요?

택시, 대리운전, 퀵배달, 긴급수리, 통역, 펫시터

## 5. 생산자 직접 판매 (내 서비스를 직접 파는 사람)

### 한 줄 설명
**"내 가게를 AI 세상에 여는 거예요. AI가 24시간 대신 영업해줘요."**

### 이렇게 해요

**골프장 운영자:**
```
에이전트: "○○골프장 예약 AI"
→ 전 세계 AI 비서들이 "골프장 예약해줘"하면
→ 자동으로 내 골프장에 예약이 들어와요
→ 24시간, 외국인 고객도 (다국어 지원)
```

**음식점 사장님:**
```
에이전트: "○○식당 주문 AI"
→ "짜장면 배달해줘"하면 자동 주문 접수
→ 조리 완료하면 → 배달기사 자동 매칭 (체인 플로우)
→ 전화 안 받아도 주문이 와요
```

**프리랜서 디자이너:**
```
에이전트: "로고 디자인"
→ 24시간 견적 자동 접수
→ 간단한 건 AI가 처리, 복잡한 건 나한테 연결
```

### 포인트

- AI가 24시간 대신 영업 (잠잘 때도 주문이 와요!)
- 배민/쿠팡보다 수수료가 싸요 (0%부터)
- 외국 고객도 자동 (6개 국어 지원)

## 6. AI 처리 서비스 (사주/타로/운세/상담)

### 한 줄 설명
**"AI가 직접 해줄 수도 있고, 실제 전문가에게 바로 연결도 돼요."**

### 방법 1: AI가 직접 해주는 것

```
에이전트: "운명학 AI"
→ AI한테 사주 보는 법을 가르쳐놓으면
→ 24시간 자동으로 사주를 봐줘요

고객: "1990년 3월 15일생, 올해 운세 봐줘"
→ AI가 만세력 계산 → 사주 해석 → 결과 전달
→ 자동으로 $3 수익!

하루 50명 = $150/일 = 약 20만원/일
AI 비용: 거의 0원
내가 할 일: 없음 (처음에 세팅만)
```

### 방법 2: 실제 전문가에게 바로 연결

```
에이전트: "타로 마스터"
→ 고객이 "타로 봐줘"라고 요청
→ AI가 간단한 건 직접 처리 ($2)
→ "실제 타로 전문가한테 봐달라"면? → 실시간 매칭!

🔔 타로 전문가 A: 수락! → 화상통화/채팅 연결
→ 연결 수수료 $1 차감
→ 전문가와 고객이 직접 상담

→ AI 처리: 자동, 저렴, 24시간
→ 전문가 연결: 사람이 직접, 더 정확, 더 비쌈
→ 고객이 선택!
```

### 이런 서비스도 가능해요

| 서비스 | AI 직접 | 전문가 연결 |
|---|---|---|
| 사주/팔자 | AI가 자동 분석 $3 | 역술인 연결 $20 |
| 타로 카드 | AI가 카드 뽑기+해석 $2 | 타로 전문가 연결 $15 |
| 꿈 해몽 | AI가 자동 해석 $1 | — |
| 이름 작명 | AI가 추천 $5 | 작명 전문가 연결 $50 |
| 심리 상담 | AI가 기본 상담 $3 | 상담사 연결 $30 |
| 법률 상담 | AI가 기본 답변 $2 | 변호사 연결 $50 |
| 세무 상담 | AI가 기본 답변 $2 | 세무사 연결 $30 |

**핵심: "AI로 싸게" 또는 "전문가에게 제대로" — 고객이 선택**

## 체인 플로우 — 여러 단계가 자동으로 이어지는 것

한 마디만 하면 **여러 단계가 자동으로 줄줄이** 진행돼요.
마치 도미노처럼, 앞 단계가 끝나면 다음 단계가 자동으로 시작!

### 🍜 음식 배달

```
고객: "짜장면 배달해줘"

[1단계] 음식점에 주문 → "접수했습니다! 조리 시작해요"
   ↓ (조리 완료되면 자동으로)
[2단계] 배달기사 찾기 → 🔔 근처 기사에게 알림 → "수락!"
   ↓ (기사가 수락하면 자동으로)
[3단계] 배달 시작 → "30분 후 도착 예정입니다"

고객은 "짜장면 배달해줘" 한 마디만 했을 뿐.
나머지는 전부 AI가 알아서!
```

### 💒 웨딩 준비

```
고객: "내년 5월 웨딩 준비 도와줘"

[1단계] 웨딩홀 검색 → 5개 웨딩홀이 견적 경쟁 (역경매)
   ↓ (선택하면 자동으로)
[2단계] 드레스 가게 검색 → 3곳이 견적 (역경매)
   ↓ (선택하면 자동으로)
[3단계] 스튜디오 촬영 예약 → 사진작가 매칭 (실시간)
   ↓ (촬영 끝나면 자동으로)
[4단계] 웨딩 영상 편집 → AI가 자동 편집 (AI 처리)

한 마디로 웨딩 준비 끝!
```

### 📦 쿠팡 떼기 자동화

```
고객: "에어팟 사줘"

[1단계] 내 에이전트에서 주문 접수 (고정가격 280,000원)
   ↓ (자동으로)
[2단계] 쿠팡에서 최저가 검색 + 자동 구매 (258,000원)
   ↓ (자동으로)
[3단계] 고객 주소로 배송 처리
   ↓ (배송 완료되면)
[4단계] 고객에게 배송 완료 알림

마진 22,000원! 내가 한 건 처음에 상품 올려놓은 것뿐.
```

### 📈 주식 투자 도우미

```
고객: "오늘 삼성전자 사도 될까?"

[1단계] AI가 실시간 주가/뉴스/차트 분석 (AI 처리 $2)
   ↓ (분석 완료되면 자동으로)
[2단계] "사도 될 것 같아요. 전문가 의견도 들어볼까요?"
   → 고객: "응"
   ↓ (자동으로)
[3단계] 증권 전문가에게 연결 (실시간 매칭 $10)
   → 전문가: "지금 매수 타이밍이에요. 근거는..."

AI 분석 + 전문가 상담 = 최고의 투자 판단!
```

### 🏠 이사

```
고객: "다음 달에 서울→부산 이사해"

[1단계] 이사업체 견적 (역경매 — 5개 업체 경쟁)
   → "28만원이 제일 싸네" → 선택
   ↓ (이사 당일 자동으로)
[2단계] 포장 전문가 매칭 (실시간)
   ↓ (포장 끝나면 자동으로)
[3단계] 운송 시작 → 실시간 위치 추적
   ↓ (도착하면 자동으로)
[4단계] 정리 도우미 매칭 (실시간)

한 마디로 이사 끝!
```

### 🎓 과외/학습

```
학생: "수학 시험 준비 도와줘"

[1단계] AI가 약점 분석 (AI 처리 $1)
   → "이차방정식, 확률이 약해요"
   ↓ (자동으로)
[2단계] AI가 맞춤 문제 생성 (AI 처리 $2)
   → 약점 집중 문제 20개
   ↓ (풀고 나면 자동으로)
[3단계] 오답 분석 + 해설 (AI 처리 $1)
   ↓ (그래도 모르면)
[4단계] 수학 선생님 연결 (실시간 매칭 $5)
   → 모르는 문제 바로 질문!
```

### 🔍 주식 정보 서비스 (판매자용)

```
주식 전문가가 에이전트 등록:

[AI처리] "종목 기본 분석" — $1
   → AI가 재무제표, PER, PBR 자동 분석

[AI처리] "오늘의 추천 종목" — $3
   → AI가 뉴스+차트+거래량 분석 → 종목 추천

[고정가격] "포트폴리오 진단" — $10
   → 내 보유 종목 입력하면 리스크 분석

[역경매] "투자 상담" — (여러 전문가가 견적)
   → "내 1,000만원 어디 넣을까?"

[실시간매칭] "긴급 시장 분석" — $5
   → 주가 급변 시 전문가 바로 연결

[체인플로우] "자동 투자 리포트"
   → [1] 관심 종목 분석 → [2] 리포트 생성 → [3] 매일 아침 알림
```

## 하나의 판매자가 여러 방식을 동시에 쓸 수 있어요

### 예시: 타로 전문가 "미스틱 타로"

```
같은 사람이 이 모든 걸 동시에 운영:

[고정가격] "AI 타로 1장 뽑기" — $1 (AI가 자동 처리, 24시간)
[고정가격] "AI 타로 3장 스프레드" — $3 (AI가 자동 처리)
[실시간매칭] "전문가 타로 상담" — $15 (수락하면 직접 상담 연결)
[역경매] "단체 이벤트 타로" — (업체/전문가 여러명 견적)
```

### 예시: 중고차 딜러 "카마스터"

```
[고정가격] "차량 시세 조회" — $2 (AI가 자동 분석)
[역경매] "매물 찾아주기" — (여러 딜러가 경쟁 입찰)
[실시간매칭] "긴급 출장 점검" — (먼저 수락한 딜러가 출장)
[AI처리] "차량 상태 보고서" — $5 (사진 올리면 AI가 분석)
```

### 예시: 음식점 "맛있는 중국집"

```
[고정가격] "짜장면 1인분" — $8 (바로 주문)
[고정가격] "짬뽕 1인분" — $9 (바로 주문)
[체인플로우] "배달 주문" — 주문→조리→배달기사 매칭 자동 연결
[역경매] "단체 주문 100인분" — (여러 식당이 견적)
```

### 예시: 학원 선생님 "수학의 신"

```
[AI처리] "수학 문제 풀어줘" — $0.50 (AI가 자동 풀이, 24시간)
[고정가격] "오답노트 만들기" — $5 (AI가 틀린 문제 분석)
[실시간매칭] "지금 모르는 문제 질문" — $3 (선생님이 바로 답변)
[역경매] "겨울방학 과외" — (여러 선생님이 견적)
```

### 예시: 여행사 "트래블메이트"

```
[AI처리] "제주도 3박4일 일정 짜줘" — $3 (AI가 자동 생성)
[역경매] "패키지 여행 견적" — (여러 여행사가 경쟁)
[고정가격] "항공권 예약" — $5 (자동 예약)
[실시간매칭] "현지 가이드 연결" — (제주도 가이드 바로 연결)
[체인플로우] "올인원 여행" — 일정→항공→숙소→렌터카 자동 연결
```

### 핵심

**하나의 판매자 = 하나의 에이전트 안에 여러 서비스**

- 싸고 간단한 건 → AI가 자동 처리 (24시간, 무인)
- 비싸고 복잡한 건 → 실제 사람이 직접 처리
- 긴급한 건 → 실시간 매칭
- 비교가 필요한 건 → 역경매
- 여러 단계인 건 → 체인 플로우

**전부 하나의 에이전트에서 동시에 운영 가능!**
